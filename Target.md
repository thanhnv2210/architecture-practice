# 🎯 Target

## Short-Term Target (MVP)

Build a static web app that can:

* Load documents from local `/docs`
* Render:

  * Markdown
  * PlantUML diagrams
* Provide a simple UI:

  * File explorer
  * Viewer panel
* Integrate with local LLM:

  * Explain architecture
  * Generate suggestions

---

## Mid-Term Target

* AI-assisted authoring:

  * Improve diagrams
  * Generate sequence diagrams
* Better UX:

  * Split view (editor + preview)
  * Dark mode
* Prompt engineering improvements

---

## Long-Term Target

* Full architecture platform:

  * Backend (Spring WebFlux)
  * Document storage (DB)
  * Version control
  * Multi-user collaboration
* AI-powered architecture analysis:

  * Detect anti-patterns
  * Suggest improvements
  * Generate test scenarios

---

## Success Criteria

* Can browse docs like a mini Notion
* Can render PlantUML reliably
* AI provides useful explanations (not generic)
* Easy to extend to backend later
